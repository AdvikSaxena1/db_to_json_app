# SQLite Database Metadata Extractor (.db to JSON Converter)

## 📌 Overview

SQLite Database Metadata Extractor is a web-based tool that allows users to upload a `.db` (SQLite database) file and automatically convert its database structure into a well-organized JSON metadata file.

The application analyzes the uploaded database, extracts important schema information such as tables, columns, data types, primary keys, and relationships, and generates a downloadable `.json` file.

This tool is useful for database analysis, documentation generation, Text-to-SQL applications, schema understanding, and AI-based database processing workflows.

---

# 🚀 Features

## 1. SQLite Database Upload

- Upload any SQLite `.db` file through the user interface.
- Automatically reads and processes the database.
- No manual database configuration required.

---

## 2. Automatic Schema Extraction

The system extracts database metadata including:

- Database tables
- Column names
- Data types
- Primary keys
- Foreign keys
- Table relationships
- Schema structure
